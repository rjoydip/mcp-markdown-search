# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [0.0.0] - 2026-06-25

### Added

- MCP server with 6 tools for markdown search
- Cloudflare Worker for semantic search and indexing
- Full-text search via ripgrep integration
- Semantic search via Cloudflare Vectorize
- Markdown file discovery and traversal
- Text chunking with heading-aware splitting
- Vector ID generation for chunk management
- Authentication via shared secret
- Documentation (usage, architecture, CLI, contributing, security)
- Testing with Bun test runner
- Linting with oxlint
- Formatting with oxfmt
- Code analysis with fallow
- Git hooks with simple-git-hooks
- CI/CD with GitHub Actions
- Deploy workflow for Cloudflare Workers
